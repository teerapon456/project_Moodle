## 1. การสร้างคำขอจองรถ (Create Booking)

### TC-001: สร้างคำขอจองรถสำเร็จ

| รายละเอียด | |
|------------|---|
| **วัตถุประสงค์** | ทดสอบการสร้างคำขอจองรถเมื่อกรอกข้อมูลครบถ้วน |
| **เงื่อนไขเบื้องต้น** | ผู้ใช้ล็อกอินเข้าระบบแล้ว |
| **บทบาท** | User |

**ขั้นตอนทดสอบ:**
1. ล็อกอินเข้าระบบด้วย User ทั่วไป
2. เข้าเมนู "รายการคำขอ" หรือกดปุ่ม "สร้างคำขอใหม่"
3. กรอกข้อมูลให้ครบถ้วน:
   - วันเวลาเริ่มต้น: เลือกวันที่ในอนาคต
   - วันเวลาสิ้นสุด: เลือกวันที่หลังวันเริ่มต้น
   - ปลายทาง: ระบุสถานที่ปลายทาง
   - วัตถุประสงค์: ระบุเหตุผลการใช้รถ
   - ผู้อนุมัติ: เลือกหัวหน้างาน
4. กดปุ่ม "บันทึก" หรือ "ส่งคำขอ"

**ผลลัพธ์ที่คาดหวัง:**
- [x] แสดงข้อความ "สร้างคำขอสำเร็จ"
- [x] คำขอปรากฏในรายการด้วยสถานะ "รอหัวหน้าอนุมัติ"
- [x] ส่งอีเมลแจ้งเตือนถึงผู้อนุมัติ (หัวหน้างาน)

## 2. การอนุมัติโดยหัวหน้างาน (Supervisor Approval)

### TC-010: หัวหน้างานอนุมัติคำขอสำเร็จ

| รายละเอียด | |
|------------|---|
| **วัตถุประสงค์** | ทดสอบการอนุมัติคำขอในขั้นตอนแรก |
| **เงื่อนไขเบื้องต้น** | มีคำขอสถานะ "รอหัวหน้าอนุมัติ" อยู่ในระบบ |
| **บทบาท** | Supervisor |

**ขั้นตอนทดสอบ:**
1. ล็อกอินเป็น Supervisor หรือคลิกลิงก์อนุมัติจากอีเมล
2. เข้าดูคำขอที่รออนุมัติ
3. กดปุ่ม "อนุมัติ"

**ผลลัพธ์ที่คาดหวัง:**
- [x] สถานะเปลี่ยนเป็น "รอผู้จัดการอนุมัติ" (pending_manager)
- [x] บันทึกวันเวลาและผู้อนุมัติในระบบ
- [x] ส่งอีเมลแจ้งเตือนถึง IPCD/Manager

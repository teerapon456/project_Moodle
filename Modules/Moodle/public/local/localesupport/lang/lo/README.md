# ພາສາລາວ - Theme Academi

ເທມ Academi ຮອງຮັບພາສາລາວ ແລະ **Moodle Boost UI** (Moodle 4.x) ແລ້ວ

## ວິທີໃຊ້

ຜູ້ໃຊ້ສາມາດເລືອກ **ພາສາລາວ** ຈາກເມນູພາສາຢູ່ມຸມຂວາເທິງໜ້າເວັບ ຂໍ້ຄວາມຂອງເທມ (ເມນູ ໜ້າຫຼັກ ຟຸດເຕີ ແຖບປະເພດ Simple UI ຫຼາຍປະເພດ ແລະອື່ນໆ) ຈະສະແດງເປັນພາສາລາວ

## Boost UI

ເທມນີ້ອີງໃສ່ Boost ຂອງ Moodle ດັ່ງນັ້ນສ່ວນໃຫຍ່ຂອງ UI (ເມນູ ປຸ່ມ drawer ຫຼັກສູດ ແລະອື່ນໆ) ໃຊ້ຮູບແບບ Boost ຄວາມສາມາດພາສາລາວຂອງເທມຈະໃຊ້ກັບຕັ້ງຄ່າເທມ ແລະ ຂໍ້ຄວາມໜ້າຫຼັກ/ຟຸດເຕີ ເພີ່ມເຕີມ

## ການຕິດຕັ້ງ Language pack

ຖ້າຕ້ອງການໃຫ້ສ່ວນອື່ນຂອງ Moodle ສະແດງເປັນພາສາລາວ ໃຫ້ຕິດຕັ້ງ Lao language pack ຈາກ **Site administration → Language → Language packs** ຫຼື [lang.moodle.org](https://lang.moodle.org)

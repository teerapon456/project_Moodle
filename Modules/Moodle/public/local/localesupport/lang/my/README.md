# ภาษาพม่า (မြန်မာဘာသာ) - Theme Academi

ธีมนี้รองรับภาษาพม่า (Myanmar) และ **Moodle Boost UI** (Moodle 4.x) แล้ว

## การเปิดใช้

1. **ติดตั้ง Language pack ภาษาพม่าใน Moodle**  
   ไปที่ **Site administration → Language → Language packs** แล้วติดตั้งภาษาพม่า (Myanmar) จาก [lang.moodle.org](https://lang.moodle.org) ถ้ามี

2. **ให้ผู้ใช้เลือกภาษาพม่า**  
   ผู้ใช้สามารถเลือกภาษาจากเมนูภาษาที่มุมขวาบนของหน้าเว็บ

3. **ฟอนต์อักษรพม่า**  
   เมื่อเลือกภาษาพม่า ธีมจะโหลดฟอนต์ **Noto Sans Myanmar** อัตโนมัติเพื่อให้แสดงอักษรพม่าได้ถูกต้อง

## Boost UI

ธีม Academi อิงจาก Boost ของ Moodle ดังนั้น UI ส่วนใหญ่ (เมนู ปุ่ม drawer รายการหลักสูตร ฯลฯ) ใช้รูปแบบ Boost ภาษาในธีมจะใช้กับตั้งค่า theme และข้อความหน้าแรก/ฟุตเตอร์/Simple UI/หลายหมวด ฯลฯ

## ข้อความในธีม

ไฟล์ `theme_academi.php` ในโฟลเดอร์นี้มีคำแปลข้อความของธีม (เมนู หน้าหลัก ฟุตเตอร์ Simple UI หลายหมวด ฯลฯ) เป็นภาษาพม่า  
ข้อความอื่นของ Moodle (เมนูหลัก หลักสูตร กิจกรรม) ใช้ language pack ภาษาพม่าของ Moodle core

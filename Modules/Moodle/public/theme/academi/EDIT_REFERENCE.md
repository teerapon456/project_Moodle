# อ้างอิงการแก้ไขธีม Academi

## สรุป: มีอะไรต้องแก้อีกไหม

สิ่งที่แก้ไปแล้วในธีมนี้:
- **db/install.php** – ไม่ error ตอนติดตั้งเมื่อไม่มีไฟล์รูปใน pix/home
- **lib.php** – ไม่ใช้ `get_url()` ของ category แล้ว สร้าง URL เอง (รองรับ Moodle 5.x)
- **classes/helper.php** – สีเริ่มต้นเป็นโทนมารูน #C02D2E
- **เทมเพลต + SCSS** – เปลี่ยน `ml-auto` → `ms-auto` (แก้ warning Deprecated .ml)
- **หน้า Login** – การ์ดมีโลโก้ + ปุ่ม Microsoft, layout กล่องกลาง/เต็มจอ
- **แถบหมวดหมู่, หลักสูตรแนะนำ, Footer, Dark mode, ฯลฯ** – ตามที่ปรับในครั้งก่อน

ถ้า deploy ไฟล์เหล่านี้ครบและ purge cache แล้ว ไม่มีอะไรต้องแก้เพิ่มสำหรับชุดฟีเจอร์ที่ทำไปแล้ว  
ถ้าต้องการเปลี่ยนข้อความ/สี/รูป เพิ่มที่ **Site administration → Appearance → Themes → Academi** ได้

## เส้นใต้ข้อความ (___) ต้องแก้ที่ไหน

เส้นใต้ที่อยู่ใต้หัวข้อหรือข้อความ มาจาก **border-bottom** ใน SCSS ถ้าต้องการเปลี่ยนสี ความหนา หรือเอาเส้นออก ให้แก้ไฟล์และบรรทัดด้านล่างนี้

### 1. หัวข้อ h2 บล็อกหน้าแรก (เช่น “หลักสูตรที่มี”, “ข่าว”)

| ไฟล์ | บรรทัด | รายละเอียด |
|------|--------|-------------|
| `scss/frontpage.scss` | 478 | `border-bottom: 3px solid $primary;` ใต้ h2 ใน #frontpage-available-course-list ฯลฯ |

- **เปลี่ยนสีเส้น:** ใช้ `$primary` อยู่แล้ว ถ้าอยากเป็นสีอื่นแทน เช่น `border-bottom: 3px solid #333;`
- **เปลี่ยนความหนา:** แก้ `3px` เป็นค่าที่ต้องการ
- **เอาเส้นออก:** ลบบรรทัด `border-bottom: ...` หรือใส่ `border-bottom: none;`

### 2. หัวข้อ h2, h3 ในเนื้อหาหน้าใน (ไม่ใช่หน้าแรก)

| ไฟล์ | บรรทัด | รายละเอียด |
|------|--------|-------------|
| `scss/content.scss` | 31 | `border-bottom: 2px solid $primary;` ใต้ #page-content h2, h3 |

- แก้สี/ความหนา/เอาเส้นออก ที่บรรทัดนี้

### 3. เส้นใต้แถบ header (แถบสีแดง/เขียวบนสุด)

| ไฟล์ | บรรทัด | รายละเอียด |
|------|--------|-------------|
| `scss/header.scss` | 3 | `border-bottom: 1px solid $color_whisper;` ของ #header |

### 4. เส้นใต้โลโก้ใน card หน้า Login

| ไฟล์ | บรรทัด | รายละเอียด |
|------|--------|-------------|
| `scss/login.scss` | 79 | `border-bottom: 1px solid $mercury;` ใน .login-card-logo |

### 5. หัวข้อในส่วนอื่น (value cards, news, footer ฯลฯ)

- **components.scss** – `.value-card-title`, `.news-org-title-block h2` (มี border/สไตล์หัวข้อ)
- **footer.scss** – h2 ในฟุตเตอร์
- **blocks.scss** – `.card-title`, h2 ในบล็อก

ถ้าพบหัวข้อไหนมีเส้นใต้แล้วไม่รู้ว่าไปจากไฟล์ไหน ให้เปิด DevTools (F12) เลือก element นั้นดู class แล้วค้นในโฟลเดอร์ `scss/` ด้วยคำว่า `border-bottom` หรือชื่อ class นั้น

## สรุปสั้น ๆ

- **“มีอะไรต้องแก้อีกไหม”** → ถ้า deploy + purge cache ครบแล้ว ไม่มีจุดบังคับต้องแก้เพิ่ม
- **“___ ใต้ข้อความ ต้องแก้ที่อะไร”** → แก้ที่ `scss/frontpage.scss` บรรทัด 478 (หัวข้อหน้าแรก) และ `scss/content.scss` บรรทัด 31 (หัวข้อหน้าใน) เป็นหลัก ส่วนอื่นตามตารางด้านบน

# ภาษาไทย - Theme Academi

ธีม Academi รองรับภาษาไทยแล้ว

## การใช้

ผู้ใช้สามารถเลือก **ภาษาไทย** จากเมนูภาษาที่มุมขวาบนของหน้าเว็บ ข้อความของธีม (เมนู หน้าแรก ฟุตเตอร์ แถบหมวดหมู่ ฯลฯ) จะแสดงเป็นภาษาไทย

## การติดตั้ง Language pack

ถ้าต้องการให้ส่วนอื่นของ Moodle แสดงเป็นภาษาไทย ให้ติดตั้ง Thai language pack จาก **Site administration → Language → Language packs** หรือ [lang.moodle.org](https://lang.moodle.org)
